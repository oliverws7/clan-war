# WarTracker - Clan War Management

WarTracker is a complete solution for Clash Royale Clan Leaders and Co-leaders to manage their clan wars, monitor member activity, and optimize performance in the River Race.

## 🚀 Features

- **Attack Monitoring**: Track who has used their 4 daily decks and who is still pending.
- **Member Management**: Identify inactive members and track performance over time.
- **Historical Data**: Analyze medal gains and participation trends.
- **Discord Integration**: Easily notify members about pending attacks.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Lucide React.
- **Backend**: Node.js, Express, MongoDB (planned).
- **Authentication**: JWT based.

## 📁 Project Structure

- `frontend/`: React application (Vite).
- `backend/`: Node.js Express API.

## 🚦 Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository.
2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```
3. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```

### Running Locally

- **Frontend**: `npm run dev` (inside `frontend/`)
- **Backend**: `npm run dev` (inside `backend/`)
